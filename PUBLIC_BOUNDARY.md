# Public Boundary

This repository contains a generic workspace framework only.

## Allowed

- architecture contracts and generic policies;
- manifest schemas and empty templates;
- synthetic examples that contain no real study content;
- generic validation code and public release documentation; and
- source-independent descriptions of optional integration points.

## Prohibited

- real project folders, data, results, manuscripts, submissions, or audits;
- patient-derived, restricted, credentialed, licensed, or confidential data;
- credentials, tokens, private paths, local runtime caches, or account details;
- copyrighted source payloads without explicit redistribution rights; and
- copied private memory, unreviewed lessons, or institution-specific claims.

## Derivation Rule

Public framework content must be independently authored or generically
rewritten, then reviewed for privacy, rights, provenance, scope, and safety.
Copying a private workspace folder into this repository is not an acceptable
derivation method.

This repository never becomes authority for a private workspace.
